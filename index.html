Projects

These are some of the projects that we are actively working on at ViDA.

ReproZip

ReproZip allows you to pack your experiment along with all necessary dependencies needed to reproduce the experiment on a different machine.

ACHE

ACHE is a focused web crawler, which means that it can automatically learn how to efficiently find and collect pages from a given domain on the Web.

Domain Discovery Tool

Domain Discovery Tool is an interactive system that helps users explore and better understand a domain as it is represented on the Web.

VisTrails

VisTrails is an open-source scientific workflow and provenance management system that supports data exploration and visualization.
